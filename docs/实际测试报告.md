### 3.2 深工勘正确版的迭代记录

深工勘样本揭示了两个真实问题，不能只看单次 error 数：

1. 末尾“本报告期间最大值”摘要列曾被映射成逐期变化速率，造成正确版误报。
2. 锚索表的“本次(kN)”实际是本期变化，“测值(kN)”才是当前内力；把测值当初始值会产生批量伪错误。

修复包括：结构化表题切块、连续续表合并、摘要表隔离、结果完整性重试、已知力值表头的确定性语义归一化，以及 `initial_value_reliable=false` 时禁止执行初始值差分。历史真实模型请求曾返回 `402 Insufficient Balance`，2026-06-21 当前源码补跑又遇到 600 秒无产物超时；2026-06-28 当前源码已完成无缓存复跑，深工勘正确版 0 error，错误版检出 5 error。
